# Full Stack Open – University of Helsinki

This repository contains my work, notes, and exercises from the **Full Stack Open** course by the University of Helsinki.

The goal of this repository is simple: to document my progress as I build a solid understanding of modern full stack web development. It will grow over time as I complete more parts of the course and refine previous work.

---

## About the Course

Full Stack Open is an in-depth, hands-on course focused on building modern web applications using JavaScript. It covers both frontend and backend development, along with testing, deployment, and scalability.

Core topics include:

* React for building user interfaces
* Node.js and Express for backend services
* REST APIs and GraphQL
* State management
* Testing (unit and end-to-end)
* TypeScript
* CI/CD and containerization

```bash
npm install
npm start
```
